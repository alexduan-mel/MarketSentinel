# News API Configuration

This document lists the API endpoints used for fetching news data.

## Massive (formerly Polygon.io)

*   **Base URL**: `https://api.massive.com`
*   **News Endpoint**: `/v2/reference/news`
*   **Documentation**: [Massive Documentation](https://massive.com/docs)

## Finnhub

*   **Base URL**: `https://finnhub.io/api/v1`
*   **General News Endpoint**: `/news?category=general`
*   **Company News Endpoint**: `/company-news`
*   **Market News Endpoint**: `/market-news`
*   **Documentation**: [Finnhub Documentation](https://finnhub.io/docs/api)
